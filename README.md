# ES2023-vs-ES2020

## Setup process

1. Clone the repo:

```
git clone https://github.com/NithinN05/ES2023-vs-ES2020.git
```

2. Install the dependencies:

```
npm install
```

## Preview Output

To preview output of es2020, run:

```
npm run es2020
```

To preview output of es2023, run:

```
npm run es2023
```
